# MAUI Keyboard Effects
Effects to make your keyboard using controls nicer.

## Features
- iOS numeric pad (vertical and horizontal layouts)
- Android numeric pad with optional horizontal layout

## Usage
```
xmlns:mke="http://maui.keyboard.effects/eightbot/2099"
```

```
<Entry mke:NumericKeyboardEffect.IsHorizontal="False">
    <Entry.Effects>
        <mke:NumericKeyboardRoutingEffect />
    </Entry.Effects>
</Entry>
```
